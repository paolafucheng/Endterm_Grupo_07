# Proyecto Vehículo - TDD

## Desarrollo de un vehículo

En este proyecto desarrollaremos de forma colaborativa una clase Vehículo utilizando Test 
Driven Development (TDD), git, GitHub y JUnit. Al finalizar, el repositorio debe reflejar buenas prácticas 
de control de versiones, tests automatizados y colaboración en equipo.

## Diseño del Vehículo (Requisitos Funcionales)

### Atributos
- **Marca**: Fabricante del vehículo (String).
- **Modelo**: Nombre del modelo (String).
